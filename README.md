# 🍳 AI Chef

AI Chef is a modern AI-powered recipe and meal planning platform built using React, Firebase, Gemini AI, and Spoonacular API.

## 🚀 Features

### 🤖 AI Recipe Generator

Generate recipes instantly using ingredients and AI.

### 🥗 AI Meal Planner

Create personalized meal plans based on:

* Calories
* Diet type
* Duration
* Language

### 📸 AI Ingredient Scanner

Upload food images and detect ingredients automatically.

### 🍽 AI Kitchen Assistant

Get recipe suggestions from available ingredients.

### ❤️ Favorites System

Save recipes for future cooking.

### 🛒 Grocery Management

* Create grocery lists
* Add ingredients automatically
* Track shopping items

### 🥫 Pantry Manager

Manage available ingredients at home.

### 📊 Nutrition Analyzer

Analyze calories, protein, fats, and carbohydrates.

### 🔍 Smart Search

Search recipes from thousands of dishes.

### 🔊 Voice Features

* Speech Recognition
* Text-to-Speech Recipe Reader

### 📄 PDF Export

Download recipes and meal plans as PDF.

## Installation

```bash
git clone https://github.com/tirupathireddy2302/ai-chef.git

cd ai-chef

npm install

npm run dev
```

---

## Environment Variables

Create a .env file:

```env
VITE_SPOONACULAR_API_KEY=YOUR_KEY
VITE_GEMINI_API_KEY=YOUR_KEY
VITE_RAPID_API_KEY=YOUR_KEY
```
